# sqb compile

Compiles all discovered models, seeds, audits, and tests. Resolves references, expands macros, validates SQL with SQLGlot, and writes compiled artifacts to `target/`.

## Usage

```bash
sqb --project-dir <path> compile [flags]
```

## Flags

| Flag | Description |
|------|-------------|
| `--no-sql-validation` | Skip compile-time SQL syntax validation |
| `--defer-to` | Resolve unselected model references against another environment |
| `--json` | Output compiled artifacts as JSON |
